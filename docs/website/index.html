Flick Player

A VFX-grade image sequence viewer. Open EXR / DPX / TIFF / PNG dailies, stack multiple layers, trim them on the master timeline, review in real OCIO color, annotate with the pen — ship the result as a video or a baked sequence. All from one fast desktop app.

Built for VFX review, not generic playback.

Flick Player reads what your pipeline produces — multichannel EXRs, OCIO-aware floats, sparse sequences with holes, multiple takes side by side — and gives you the tools you need to actually review them. Stack, trim, annotate, comment, export, share.

Real-time playback

Async RAM cache + OpenGL viewport. Smooth scrubbing on 4K EXRs at 24 fps. Worker pool auto-tunes to your CPU.

Multi-layer stack

Stack takes, mattes, plate & comp side by side. Reorder by drag, trim each layer's IN/OUT on the timeline, slide its offset.

OCIO color

ACES 2.0 CG built-in. Source / display / view round-trip on the GPU shader — toggling is free.

Annotations

Pen / eraser / 7-color palette / per-frame undo. Persistent strokes — and a Google-Meet-style ghost mode.

Multichannel EXR

Channel selector for AOVs, depth, normals, cryptomattes — per layer. Per-channel R/G/B/A mute toggles.

Smart cache

Reload only re-decodes files whose mtime changed. Missing frames serve a rich placeholder so playback never freezes.

Undo / Redo

Edit menu & Ctrl+Z / Ctrl+Y revert layer add / remove / reorder, drop-replace, trim and offset edits — atomic per gesture.

Export

PNG / JPG / EXR / TIFF sequences or H.264 / H.265 / ProRes / DNxHR / VP9 / FFV1 / v210 video — annotations baked in.

Comments

Per-frame text threads beside the visual annotations. Author auto-filled from OS user, blue dot on the timeline.

Sessions

Save the full layer stack — sources, offsets, trims, channels, alpha modes — to a single .session JSON file.

Sidecar JSON

Annotations + comments live in one human-readable JSON file next to the sequence. Travels with the work.

Hardware-adaptive

Auto-tunes cache budget + worker count to your GPU / RAM. PBO upload on discrete GPUs, sync upload on integrated.

Layers, trim and offset.

Drop multiple sequences and Flick Player stacks them. Each layer carries its own offset on the master timeline, its own IN / OUT trim, its own channel selection, its own alpha-composite mode. Drag the body to slide, drag a handle to trim — the right-edge stays put on IN-trims, just like a real NLE.

  • Drop on the viewer = replace the focused sequence. Drop on the panel = add as a new layer above.
  • Drag the row vertically to reorder. A drop indicator shows where it'll land.
  • Trim handles snap to the playhead, master IN/OUT and other layers' edges.
  • Ghost overlay (dashed orange) shows source frames hidden by the trim — drag the handle back outwards to reveal them.
  • Per-layer T / αS toggles control alpha-over compositing and straight-vs-premultiplied alpha handling.

A timeline that tells you what's going on.

One bar carries every dimension of the sequence's state — at a glance, you know which frames are cached, which are missing, which fall in a multi-layer gap, where you've left a note, what's in/out range, and where the playhead is.

  • Translucent orange runs = frames in RAM. The bar shows your prefetch high-water mark.
  • Mid-grey runs = no layer covers — Flick plays through and the viewport paints a checkerboard.
  • Green triangles = annotated frames. Blue dots = frames with comments.
  • Ctrl + click sets and drags the in / out points — like Nuke.
  • The cursor scrubs outside the in/out range; playback snaps back inside on press.

Two ways to annotate.

Persistent = strokes saved to a sidecar JSON, frame-keyed, ready to share. Ghost mode (👻) = strokes fade out on their own, perfect for live screen-share commentary in Meet / Discord. The toolbar's outer border turns cyan when ephemeral mode is active so the state is unmistakable.

  • 7-color review palette tuned for VFX (red / yellow / green / blue / orange / b&w).
  • Per-frame undo — Ctrl+Z only undoes the current frame's edits, no surprise jumps.
  • Eraser deletes whole strokes by hit-testing the polyline, not pixel-level.
  • Three fade presets in ghost mode: court ~2 s, moyen ~5 s (default), long ~10 s.
  • 👁 Show during playback — toggle visibility in real time without losing your work.
  • Bake into export if you want a self-contained MP4 to send a colleague.

Undo / Redo for everything you touch.

The Edit menu and the Ctrl+Z / Ctrl+Y shortcuts revert layer add, remove, reorder, drop-replace, IN / OUT trim and body offset edits — atomic per gesture. A multi-step replace (= remove every existing layer + add the new one) collapses to a single undo entry so a misclick is one tap to recover.

  • Layer mutations — add / remove / reorder / toggle visible.
  • Drop replace on the viewer area — one undo, restores the previous stack as it was.
  • Drop add-layer on the panel area — one undo removes the new top.
  • Trim & offset commits emitted on mouse release, so a drag is one undo entry.
  • History depth capped at 100; oldest entries fall off without warning.

Real OCIO color, on the GPU.

Flick Player ships with the ACES 2.0 CG built-in config. Point it at your studio's custom config.ocio via the $OCIO env var if you have one. The display transform runs on the fragment shader, so toggling source / display / view is free.

  • Auto-detect input colorspace from filename + EXR header attributes.
  • Per-image exposure nudge with + / - (±0.25 stops).
  • Per-layer channel selection — pick a single AOV / depth / normals layer and view it as RGB.
  • RGBA mute: toggle channels off in the shader without invalidating the cache.

Per-frame comments for the team.

Annotations are visual; comments are textual. A side panel beside the viewer holds a thread of notes attached to whatever frame the playhead is on — perfect for "the lighting is too cool" feedback that doesn't need a stroke.

  • Author auto-filled from the OS username — no login flow.
  • Created / edited timestamps tracked per comment.
  • Blue dots on the timeline mark every commented frame for one-click jumps.
  • Same JSON sidecar as annotations — one file travels with the sequence.
  • The Comments tab shares the side dock with Color; flip with one click.

Export to anything.

Send your sequence as an image-per-frame run for the next tool in the pipe, or bake it into one of eight video codecs to ship a review reel. Annotations toggle on / off; the OCIO display transform too. Three policies for sequences with holes, and an aspect-ratio lock for custom resolutions.

  • Range: full sequence, in/out, or custom from / to.
  • Resolution: source / 720p / 1080p / 4K UHD / custom (with aspect lock).
  • Frame rate: source / 23.976 / 24 / 25 / 29.97 / 30 / 60 / custom.
  • Color: bake the OCIO display transform, or keep the linear buffer untouched.
  • Annotations: bake into pixels + optional sidecar JSON copy alongside the export.
  • Missing-frame policy (see below): never let a hole abort an overnight render again.

Three policies for missing frames.

Found a hole in the sequence at 03:12 AM during the overnight render? Pick the policy on the way out so the export decides what to do without you.

  • Abort — fail loud at the first missing frame. Default; safest for finals.
  • Black — substitute a black frame. Quick when the gap is near a cut and you're shipping a draft.
  • Placeholder — write the rich "MISSING FRAME" graphic. Best for a dailies pass — reviewers can't miss the issue.

Save the whole stack as a session.

File → Save session writes every layer's source path, offset, IN/OUT trim, channel selection, alpha-composite mode and visibility into one .session JSON file. Reopen later and the stack reappears exactly as it was. Travels next to the work; readable by hand if you ever need to check what's in there.
